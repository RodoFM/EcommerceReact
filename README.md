# React + Vite

Esta es una app ecommerce, realizada para trabajo de clase de React. No requiere instalación. La página esta orientada para pequeña pyme en la cual se realizarán ventas reales en el futuro.
Contiene varios componentes, enlazada con firebase para los products, y con cloudinary para el upload de imágenes para los productos.
Creada con React+Vite.
Posse un pequeño cátalogo con productos con stock, que ha medida que se compre baja el sock. Además, un carro de compra para visualizar aquello que se comprará.
